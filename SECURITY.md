# Security Policy

## Supported Versions

Only the latest live version of the bot is supported.  
This refers to the current production instance available through the [official invite link](https://discord.com/oauth2/authorize?client_id=1450060292716494940).

## Reporting a Vulnerability

If you discover a security vulnerability, please report it to me (MuhamDaily) at <security@muhamdaily.dev> or private message on [Discord](https://discord.gg/wYZa4Fpwfy).

Reports are typically acknowledged within 3 days.

I will make a reasonable effort to:
- Keep you informed about the progress of the investigation and fix
- Coordinate disclosure where appropriate
- Request additional details if needed to reproduce or verify the issue

> **Note:** Ensnare is a personal project and does not generate revenue, so I cannot guarantee extensive rewards or a formal bug bounty. Responsible disclosure is greatly appreciated.
